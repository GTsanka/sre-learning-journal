## What I learned
- WSL runs Linux inside Windows, no dual-boot needed
- Prompt format: user@host:path$ — $ = normal user, # = root
- sudo apt update refreshes package list, sudo apt upgrade installs updates
- Linux commands are case-sensitive: ssh -t and ssh -T do different things
- Paths use / as separator; ~ means home directory
- SSH keys authenticate me to GitHub without typing passwords
